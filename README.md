## 📌 Overview

This project is an end-to-end **AI-based Speech-to-Indian Sign Language (ISL) Translator** designed to bridge the communication gap between spoken language users and the hearing-impaired community.

It combines **Speech Recognition, Natural Language Processing (NLP), and Computer Vision** to convert speech into text and recognize corresponding sign language gestures.

---

## 🚀 Features

- 🎤 Real-time speech-to-text conversion  
- 🧹 Text preprocessing using NLP techniques  
- 🧠 CNN-based hand gesture recognition  
- 📷 Real-time detection using webcam  
- ⚡ Fast and efficient predictions  
- 🔄 Integrated pipeline for speech + gesture processing
